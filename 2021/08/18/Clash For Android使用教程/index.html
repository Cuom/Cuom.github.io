Clash For Android使用教程

发表于2021-08-18|更新于2023-05-24

简介

Clash 是一个使用 Go 语言编写，基于规则的跨平台代理软件核心程序。

Clash for Android 是安卓系统上的一款 Clash 客户端。

支持的协议：Vmess, Shadowsocks, Snell, SOCKS5, ShadowsocksR

Github 项目地址：https://github.com/Kr328/ClashForAndroid

Clash for Android 为免费 app ，已于 2019.12.10 上架 Google Play 。

添加订阅

Clash for Android 支持两种导入配置文档的方式：

  • URL （订阅）
  • 本地导入

通常情况下使用订阅链接导入即可。

点击配置 在新弹出的窗口中点击右上角+。

选择从URL导入。在对应地方填写订阅地址并保存。

启动服务

选择新添加的订阅配置

回到首页点击 启动

切换节点及延迟测试

在开启代理后，可以点击中间的 代理

进入策略组面板，在这里可以切换节点

直接点击你想要的节点即可进行切换。

点击⚡图标可以进行延迟测试，测试结果将显示在节点名称右侧。

更改代理模式

点击右上角的 ⋮ 按钮进行更改代理模式

首页点击 设置 覆写 模式 进行更改代理模式

更改代理模式需同时操作。

规则 模式是根据默认规则列表进行代理。
直连 模式是所有连接不通过代理。
全局 模式下所有连接全部通过代理。

语言设置

语言设置路径：Settings → Interface → Language → Simplified Chinese
设置 👉 界面 👉 语言 👉 简体中文

暗黑模式

入口：设置 👉 界面 👉 暗黑模式

省电策略 / 允许后台运行

长按 Clash 图标，选择应用程序资讯
勾选允许自动启动和允许其他应用程序关联启动
点击耗电保护 , 选择允许背景执行
