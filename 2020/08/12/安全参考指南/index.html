安全参考指南

发表于2020-08-12|更新于2023-05-22
|阅读量:

系统安全篇

  1. 请检查系统是否安装有 360 金山毒霸 等国产安全软件，在科学上网时请彻底关闭此类软件（一定要彻底关闭，最好前往任务管理器确认是否有留存进程）。
  2. 上网时请不要在某些特殊网站点击任何广告，更不要赋予任何网站以高级的权限（特如定位权限）
  3. 请给电脑/手机加上一把硬件锁，请使用类似Bitblock/TPM-Luks的硬盘加密手段保护你的个人硬盘数据。
  4. 可以为手机/电脑开启来宾系统( Guest OS )，相互有不同的密码，各不互通（最好是有两套不同的设备，一套安装国产软件，一套不沾任何国内软件）

软件安全篇

  1. 代理软件请尽量开启 TUN/TAP ，以避免DNS/WebRTC侧漏你的真实IP。
  2. 请使用最新的，官方的，非国产，与国内无往来业务的 Chrome / Chromium / Firefox 系浏览器，以避免出现安全漏洞/泄露个人隐私（特指360安全浏览器，双核高速浏览器，Firefox CN，Microsoft Edge等浏览器）
  3. 可以使用Adguard等软件拦截广告追踪，跟踪参数等泄露隐私的行为。
  4. 手机端代理软件可以开启根据应用分流，以防止被检测出代理行为。

行为安全篇

  1. 请不要在任何公开的社交平台透露你会科学上网/正在科学上网等有关行为，注意在国内平台的言行举止，切忌将国内聊天软件的你与境外聊天软件的你相关联（例如昵称，个人信息，说话风格，密码等特征）！
  2. 请不要在任何不信任的软件（包括 Google Twitter 类平台）开启/透露你的定位，以避免安全问题。
  3. 请不要使用国内手机号/邮箱注册国外服务 （特指+86手机号,QQ,163等邮箱）
  4. 请尽量使用 DNS over HTTPS/TLS/QUIC 等DNS协议（推荐使用Adguard），注意访问网址是否为 HTTPS:// 开头，若没有，可以手动加上 S。
